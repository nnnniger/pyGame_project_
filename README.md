# pyGame_project_
О нашем проекте
 Наша игра - “Легенда Сражений: Пиксельная Битва в Космосе”, аркадный платформер с открытым миром, заданиями и еще кучей всего интересного.Введение игры начинается с предыстории, которая служит начальным экраном. В этой предыстории представлены два главных героя - маленький робот-исследователь и космонавт. Космонавт отправляется в неизведанный космический сектор на свою миссию.
Задача космонавта - исследовать два разных планетных уровня и собрать ценные артефакты. Эти артефакты помогут раскрыть тайну этого космического сектора и узнать его историю. 
После успешного прохождения первого уровня игрок получит достижение и перед началом второго уровня ему будет представлен выбор за какого из персонажей играть - маленького робота-исследователя или космонавта. Оба персонажа будут иметь лазерную пушку в качестве своего оружия.
При запуске программы открывается главное окно, где игрок может ввести свое имя. Рядом с именем находится кнопка, которая позволяет посмотреть таблицу лидеров - это список лучших игроков. Ниже расположены две кнопки - "Story" и "Endless", с помощью которых игрок может выбрать желаемый режим игры."Story" - это режим, в котором игрок сможет продолжить прохождение истории, состоящей из двух уровней. "Endless" - представляет собой режим с бесконечным появлением монстров волнами. В этом режиме игрок сможет бороться с монстрами до тех пор, пока не будет повержен.
Таким образом, игра предлагает захватывающий сюжет, возможность выбора персонажа и два увлекательных режима игры, чтобы игрок мог насладиться захватывающими космическими приключениями.
Управление в данной игре осуществляется с помощью клавиш на клавиатуре. Для передвижения персонажа вы можете использовать клавиши "W", "A", "S" и "D". Клавиша "W" отвечает за движение прыжок, "A" - движение влево, "S" - движение не знаю, а "D" - движение вправо. Нажав на клавишу "Пробел", персонаж выстрелит.
